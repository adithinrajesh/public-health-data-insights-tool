# Public Health Data Insights Tool

A web-based tool to analyze, summarize, filter, and visualize public health data.

---

## Features

- Patient Data Management: Add, update, delete, and view patient records.
- Filtering: Apply multiple filters on patient data (age, gender, hospital, medical condition, billing amount, etc.).
- Summarization: Generate group-by and aggregate summaries with mean, min, max, and count.
- Curated Visualizations: Predefined charts for demographics, admissions, billing, test results, and more.
- Export: Download filtered or summarized data as CSV.
- Logging: Tracks user activities and system events.

---


## Getting Started

### Prerequisites

- Python 3.10+ (Recommended 3.14-64 as used in development)
- pip (Python package installer)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/adithinrajesh/public-health-data-insights-tool.git
cd public-health-data-insights-tool

2. Install Dependencies:

pip install -r requirements.txt

3. Run the application:

python run_flask.py

4. Open http://127.0.0.1:5000/ in your browser
